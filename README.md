# Endeavour Bootstrap

A collection of somewhat opinionated bootstrapping scripts for kitting out a fresh install.

Collated for use with [Endeavour OS](https://endeavouros.com/) but it should work with any Arch-based OS. 

Basic, but effective.

## `default`
- Mise
- OhMyBash
- Git
- Docker
- Docker Compose
- cURL
- jq
- Spotify
- Slack
- VSCode
- Lazydocker
- Claude
